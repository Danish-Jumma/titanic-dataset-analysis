# 🚢 Titanic Pandas EDA Project

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using **Pandas** to understand survival patterns based on passenger attributes.

---

## 🎯 Objective
To analyze how features like gender, class, age, and fare influenced survival chances.

---

## 🛠️ Tools Used
- Python
- Pandas
- Jupyter Notebook

---

## 📊 Dataset
The dataset is loaded from Seaborn's built-in Titanic dataset.

---

## 🔍 Key Analysis Performed
- Data cleaning (handling missing values)
- Survival analysis by gender
- Survival analysis by passenger class
- Combined analysis (gender + class)
- Age and fare distribution analysis

---

## 📈 Key Insights
- Females had higher survival rates than males
- First-class passengers had better survival chances
- Third-class passengers had lowest survival rate
- Age distribution shows most passengers were young adults
- Higher fare correlated with higher survival probability
